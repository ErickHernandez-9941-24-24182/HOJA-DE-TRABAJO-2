Descripcion:

Este programa crea tres nodos dinámicamente usando new, donde cada nodo almacena un producto (código, nombre y precio).
Los nodos se enlazan manualmente formando una lista simplemente enlazada. Luego, se recorren utilizando un puntero auxiliar para imprimir los datos almacenados.
Finalmente, se libera correctamente la memoria con delete, evitando fugas de memoria.
